# Examples

## Legend

{S … }  subject frame (solid white frame; used only when a subject role is established)  
{V … }  verb frame (flowing white frame; predicate head)  
{O … }  object frame (pulsating white frame; direct object head)

( … )   lexical element or modifier (unframed)  
|       pause / darkness (group boundary; establishes clause structure)

---
